## Responsive Design Principles

### 1. Grid System

| Screen Size | Columns | Gutter | Max Width |
|-------------|---------|--------|-----------|
| Mobile | 1 | 16px | 100% |
| Tablet | 2 | 20px | 720px |
| Desktop | 3 | 24px | 1200px |

### 2. Typography Scaling

| Element | Mobile | Tablet | Desktop |
|---------|--------|--------|---------|
| Page Title | text-2xl | text-3xl | text-3xl |
| Section Title | text-xl | text-xl | text-2xl |
| Card Title | text-lg | text-lg | text-xl |
| Body | text-base | text-base | text-base |
| Labels | text-sm | text-sm | text-sm |
| Data | text-sm | text-base | text-base |

### 3. Spacing Scale

| Context | Mobile | Tablet | Desktop |
|---------|--------|--------|---------|
| Page padding | 16px | 24px | 32px |
| Section gap | 16px | 24px | 32px |
| Card padding | 16px | 20px | 24px |
| Component gap | 12px | 16px | 16px |

### 4. Touch Targets

| Element | Mobile | Desktop |
|---------|--------|---------|
| Buttons | 48x48px min | 36x36px min |
| Table rows | 56px min | 44px min |
| Cards | Full width | Variable |
| Form inputs | 48px min | 36px min |

## Component Breakpoint Behavior

### Cards

| Breakpoint | Behavior |
|------------|----------|
| Mobile | Full width, stacked vertically |
| Tablet | 2 per row, wrap |
| Desktop | 3 per row, wrap |

### Tables

| Breakpoint | Behavior |
|------------|----------|
| Mobile | Horizontal scroll or collapsible rows |
| Tablet | Full display with reduced spacing |
| Desktop | Full display |

### Charts

| Breakpoint | Behavior |
|------------|----------|
| Mobile | Full width, simplified tooltips |
| Tablet | Full width, enhanced tooltips |
| Desktop | Full width, full interactivity |
