## üé® DESIGN PRINCIPLES

### Core Principles
1. **Less is More** - Remove everything unnecessary
2. **Form Follows Function** - Design for usability
3. **Consistency** - Use design system, don't reinvent
4. **Accessibility** - Design for everyone
5. **Simplicity** - Complex problems, simple solutions

### Visual Design
- Generous whitespace
- Clear visual hierarchy
- Limited color palette (use system)
- Consistent spacing (8px grid)
- Readable typography

### Interaction Design
- Clear feedback (user knows what happened)
- Predictable responses (no surprises)
- Forgiving (easy to undo/fix mistakes)
- Efficient (fewest steps to goal)
- Learnable (discoverable patterns)
